# TheraAssist AI Frontend

The React + TypeScript user interface for the TheraAssist AI academic prototype.

## Commands

```bash
npm install
npm run dev
npm run build
npm run lint
```

During local development, Vite proxies `/api/*` requests to the Flask backend at `http://localhost:5000`.

For project-level setup, architecture, limitations, and attribution, read the root [`README.md`](../README.md) and [`NOTICE.md`](../NOTICE.md).
